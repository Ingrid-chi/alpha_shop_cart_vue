# alpha-shop-cart-vue

建立 ALPHA-Shop 購物車的購物流程、購買商品數量、及總價格。

##功能列表

- 購物車步驟圖，可自由切換所執行到的步驟，畫面也會跟著步驟變動。
- 表單填寫，文字輸入與下拉是選單
- 運送方式選項
- 購物籃裡的數量增減按鈕
- 總價格包含商品價格與運費
